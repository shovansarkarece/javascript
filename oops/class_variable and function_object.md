# Class Variable and Function Object
```
class Person={
  first_name:"Suva"
  last_name:"Sarkar"
  age:100
  isBangladesh:true
  getName:()=>{
    return `My First name is ${person.first_name} ${person.last_name}`
}
}
let Person= new Person();
console.log(Person.first_name)
console.log(Person.age)
```
